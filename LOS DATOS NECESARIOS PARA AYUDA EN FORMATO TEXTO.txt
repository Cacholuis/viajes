PASOS PARA LLEVAR BASE DE DATOS A OTRA PC PARA TRABAJAR EN LA NUBE
En la terminal de VS Code (ya fuera del prompt de MySQL), ejecuta:
bash
mysqldump -u root -p --default-character-set=utf8 viajes > E:\respaldoviajes.sql
o este 
mysqldump --default-character-set=utf8mb4 -u root -p viajes > E:\respaldoviajes.sql

YA EN LA PC ¿
Abre la terminal y crea la base de datos vacía:
mysql -u tu_usuario -p -e "CREATE DATABASE nombre_base;"
Restaura el respaldo:
bash
mysql -u tu_usuario -p nombre_base < respaldo.sql
Verificar que se restauró bien
Entra al prompt de MySQL:
bash
mysql -u tu_usuario -p
Luego ejecuta:
sql
USE viajes;
